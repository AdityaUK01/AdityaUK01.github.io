End-to-end ETL pipeline extracting retail sales data from MySQL, transforming with Python and Pandas, loading to AWS S3 with date-based partitioning. Airflow DAGs orchestrate daily batches with retry logic.

  • Processing 10,000+ records per daily run with full schema validation
